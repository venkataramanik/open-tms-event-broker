# Open TMS Event Broker

A robust, 100% vendor-agnostic infrastructure blueprint for high-throughput transactional logistics data. Built completely with open-source tools to handle asynchronous ingestion, message queuing, and spatial-ready persistence layers.

## 🛠️ The Architecture
* **API Gateway:** FastAPI
* **Event Stream Broker:** RabbitMQ
* **Persistence Engine:** PostgreSQL with PostGIS extensions
* **Execution Layer:** Custom Native Python Workers
